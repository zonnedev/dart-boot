# boot_http_generator

Code generator for Boot HTTP controllers (`@Controller` routes).

Part of the [Boot Framework](https://github.com/zonnedev/dart-boot).

## Usage

Add as a dev dependency:

```yaml
dev_dependencies:
  boot_http_generator: ^0.1.0
  build_runner: ^2.4.0
```

The generator auto-applies to any package that depends on `boot_http`.
